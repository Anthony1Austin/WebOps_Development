# Roofing Website Template

A specialized template for roofing contractors and construction companies. Includes service area focus, project galleries, and lead generation forms optimized for local service businesses.

## Roofing-Specific Features

- ✅ **Service Areas** - Highlight service locations
- ✅ **Project Gallery** - Before/after photos
- ✅ **Service Types** - Different roofing services
- ✅ **Free Estimates** - Lead generation forms
- ✅ **Emergency Service** - 24/7 availability
- ✅ **Insurance Claims** - Help with insurance
- ✅ **Financing Options** - Payment plans
- ✅ **Local SEO** - Optimized for local search

## Local SEO Optimization

### Key Elements
- Service area pages
- Local business schema
- Google Business Profile integration
- Local citations
- NAP consistency (Name, Address, Phone)
- Location-specific content
- Local keywords

### Local Business Schema
```json
{
  "@type": "LocalBusiness",
  "name": "Your Roofing Company",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "123 Main St",
    "addressLocality": "City",
    "addressRegion": "State",
    "postalCode": "12345"
  },
  "telephone": "+1-XXX-XXX-XXXX",
  "areaServed": ["City 1", "City 2", "City 3"]
}
```
