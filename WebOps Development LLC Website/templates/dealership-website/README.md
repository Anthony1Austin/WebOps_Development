# Dealership Website Template

A specialized template for car dealerships, motorcycle shops, and vehicle retailers. Includes inventory management features and vehicle-specific schema markup.

## Dealership-Specific Features

- ✅ **Inventory Showcase** - Display vehicles with filters
- ✅ **Vehicle Details** - Comprehensive vehicle information
- ✅ **Search & Filters** - Find vehicles by make, model, price, etc.
- ✅ **Financing Calculator** - Help customers calculate payments
- ✅ **Trade-In Form** - Accept trade-in offers
- ✅ **Service Department** - Schedule service appointments
- ✅ **Parts & Accessories** - Showcase parts inventory
- ✅ **Vehicle Schema** - Rich snippets for vehicles

## Customization Guide

### 1. Inventory Management
- Vehicle listing page with grid/list view
- Individual vehicle detail pages

### 2. Vehicle Schema Markup

{
