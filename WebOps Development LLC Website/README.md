## Customization

### Update Company Information

1. **Contact Information** - Update in `index.html`:
   - Email: `info@webopsdev.com`
   - Phone: `+1 (XXX) XXX-XXXX`
   - Address (if applicable)

2. **Social Media Links** - Update in structured data (JSON-LD) in `index.html`

3. **Brand Colors** - Modify in `css/variables.css`:
   - Primary color (currently blue)
   - Secondary color (currently teal)
   - Accent color (currently orange)

### Add Real Template Screenshots

Replace placeholder images in `js/templates.js` with actual screenshots:
1. Take screenshots of each template
2. Save to `assets/images/templates/`
3. Update image URLs in `templates.js`

### SEO Configuration

Before launching:
1. Update all meta tags in `index.html`
2. Update `sitemap.xml` with your actual domain
3. Update `robots.txt` with your domain
4. Add Google Analytics (uncomment in HTML)
5. Submit sitemap to Google Search Console
